You can add the following options to the JSON data to customize the generated page: 

  • pageTitle
    The title of the generated page. 
  • pageContent
    The main content of the page. It consists of header, main, and footer sections. 
  • header
    The header section of the page. Contains the title and possibly other elements like a navigation bar. 
    • elementType
      The type of element, for example, "header". 
    • text
      The text content of the header. 
    • textColor
      The color of the text in the header. 
    • backgroundColor
      The background color of the header. 
  • main
    The main content section of the page. It can contain multiple articles or sections with their respective content. 
    • elementType
      The type of element, for example, "main". 
    • articles
      A list of articles or sections in the main content. 
  • footer
    The footer section of the page. Usually contains copyright information, contact details, etc. 
    • elementType
      The type of element, for example, "footer". 
    • text
      The text content of the footer. 
    • textColor
      The color of the text in the footer. 
    • backgroundColor
      The background color of the footer.

Additionally, each article or section in the main content can have the following options: 

  • sectionTitle
    The title of the article or section. 
  • sectionContent
    The content of the article or section. This can be a list of divs, images, or other elements. 
  • layout
    The layout of the section content. Possible values are "stacked" (elements are stacked vertically) and "flex" (elements are displayed in a row).

Each div in the section content can have the following options:

  • elementType
    The type of element, for example, "div" or "img". 
  • content
    The content of the div (text or HTML). 
  • textColor
    The color of the text in the div. 
  • backgroundColor
    The background color of the div.

Additionally, if the element type is "img", it can have the following options: 

  • src
    The URL of the image. 
  • alt
    The alt text of the image.
